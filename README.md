## Installation
1. Clone the repository:
``` sh
git clone https://github.com/tyanfarm/LGATFedRec
```

2. Open folder: 
``` sh
cd LGATFedRec
```

3. Install the dependencies:
``` sh
pip install -r requirements.txt
```

4. Create logs folder
``` sh
mkdir log sh_result
```

## Start training
To start training, use the following command:

``` sh
python train.py
```
